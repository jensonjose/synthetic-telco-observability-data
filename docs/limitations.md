# Limitations

I want this project to be useful, but I also want the boundaries to be clear.

- The data is synthetic and simplified.
- It does not capture the full complexity of telecom operations.
- It is not a real NWDAF.
- It is not a real OSS, BSS, EMS, NMS, or service assurance platform.
- It is not a real network simulator.
- It does not model full 3GPP or O-RAN behavior.
- It should not be used for production operations.
- Scenario propagation is intentionally compact and deterministic.
- Coordinates, customer segments, service names, topology, and identifiers are fabricated.

The best use of this repo is experimentation: RCA prompts, anomaly detection, analytics prototypes, blog examples, and closed-loop automation labs.
